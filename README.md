# FamilyCalendar
Gathers events and appointment from various calendar apps of multiple persons and generates a picture with all of them

### Goals:
* Integrations:
    * Google ✓
    * WebUntis ✓
    * Outlook
    * Apple Calendar
    * ical

* Picture Creation
    * Standalone
    * Magic Mirror Module
    * epaper compatible

* Misc
    * Telegram Notification/Daily Reminder Bot

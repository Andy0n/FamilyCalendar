# FamilyCalendar
Gathers events and appointment from various calendar apps of multiple persons and generates a picture with all of them

### Goals:
* Integrations:
    * Google Calendar ✓
    * WebUntis ✓
    * Outlook Calendar
    * Apple Calendar
    * ical, ics

* Picture Creation
    * Standalone ✓
    * Magic Mirror Module
    * epaper compatible

* Misc
    * Telegram Notification/Daily Reminder Bot
    * Creation of an ics Calendar
